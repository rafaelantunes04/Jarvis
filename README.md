# APP JARVIS
Personal Assistant Application that focuses on project building and evolving over time, totally local or hosted at home.

### Setup Repo and Run Application
1. Open terminal
2. `pip install uv`
3. `uv sync`
4. `uv run dev`

### Setup Local Chatbot
1. Install Ollama
2. Open terminal
3. `ollama pull llama3.2:1b`

## Setup and Run Aplication
1. Install Rust Programming Language
2. Make sure its up to date
3. Open terminal
4. Make sure the application is running (`uv run dev`)
5. `cargo tauri dev`
   
## Run Simple Chat CLI
To run, while the application is up (`uv run dev`), run the tests/simple_chat_cli.py.
